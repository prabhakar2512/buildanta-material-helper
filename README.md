# Buildanta Material Helper

## Overview

Buildanta Material Helper is an AI-powered web application that helps first-time homeowners get guidance before purchasing construction materials.

## Features

* Material input field
* Quick-select buttons for Cement, Tiles, and TMT Bar
* AI-generated recommendations using Gemini AI
* Error handling
* Responsive UI

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Google Gemini API

## Installation

```bash
npm install
npm run dev
